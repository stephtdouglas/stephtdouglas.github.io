Data & Code


 

The Factory and the Beehive II:
Rotation and Activity in Praesepe and the Hyades
Douglas et al. 2014, ApJ, 795, 161

The catalogs for the Praesepe and Hyades clusters are available as extended tables through the journal.

A tab-separated version of Table 8 from the paper, giving synthetic colors and chi values (for converting from Halpha equivalent widths to luminosity ratios), is here

In Section 4.3 we fit a saturation-type model to Halpha activity vs. Rossby number using the open-source MCMC code emcee. The code I wrote for that fit is on GitHub, and samples from the posterior distribution are available here


 

Fitting synthetic spectra to data

For my project studying L dwarfs, I'm developing an open source code for fitting synthetic spectra from atmosphere models to observed data. The code is currently called bdmcmc and is available on GitHub.
